## Commands

### To run the project on a computer
```sh
# Clone the project
git clone https://github.com/ironhack-dev-squad-127/celeb
cd celeb

# Install the dependencies
npm install

# Start the project on http://localhost:3000
npm start
```

### Other commands

```sh
# To build the project
npm run build
```
